# Smart_Clean_City

Smart Clean City is a web-based application designed to digitally manage city cleanliness and waste management systems.
The main objective of this project is to create a transparent and efficient communication platform between citizens and municipal authorities.
# Key features

CITIZEN MODULE
* User registration and login system
* Online submission of cleanliness complaints
* Real time complaint status tracking
* Location based issue reporting

WASTE MANAGEMENT
* Area wise cleanliness monitoring
* Garbage collection tracking
* 
SYSTEM FEATURES
* Secure authentication and authorization
* Data stored safely in the database
* Accessible on both desktop and mobile devices
